# Moim
- Demo: http://default-environment.46vpqtvh3w.ap-northeast-2.elasticbeanstalk.com/


##설명
개인 성격 분석을 통한 모임 추천이 가능한 온라인 모임 시스템

##기능
- 온라인 모임 생성 및 관리
- 온라인 만남 생성 및 관리
- 만남 참여자들의 위치 파악을 통한 만남 장소 추천
- 개인 성격 분석을 통한 모임/만남 추천
- 개인 성격 분석을 통한 모임 생성 추천


##사용 기반 기술
- Spring Framework
- Spring boot
- Spring Data REST
- JPA
- Vaadin

##배포 환경
- AWS Beanstalk
